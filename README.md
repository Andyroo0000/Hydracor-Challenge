# Hydracor-Challenge

a) I decided to create a command line app. All you need to run it is Python Installed and then run the program on terminal

b) The features I implemented were first adding media items. You can choose to first add one in which you need to type in a book, movie, anime, or game. Then you can enter the name of either of them, the status in percentage of completion, how well you rate it, and personal notes. Next, you can view all the different media items. Also, you can sort and filter your media items. You can filter it by media type such as game, movie, anime, or book. You can also sort it by alphabetical order, rating from best to worse, and completition. In addition, you can remove items by typing the media item name. Lastly, there is a save and load function that uses json files. It automatically loads when you start the program and saves everytime you enter an action.
