# Hydracor-Challenge
a) I decided to create a command line app. All you need to run it is Python installed, then run the program in the terminal. The file to run is called challenge.py

b) The features I implemented were first adding media items. You can choose to add one by typing in a book, movie, anime, or game. Then you can enter the name, the status as a percentage of completion, a rating, and personal notes. Next, you can view all the different media items. You can also sort and filter your media items by media type such as game, movie, anime, or book. Additionally, you can sort by alphabetical order, rating from best to worst, and completion percentage. You can also remove items by typing the media item's name. Lastly, there is a save and load function that uses JSON files. It automatically loads when you start the program and saves every time you enter an action.

c) I think a cool feature to add would be integrating a database of books and pulling from that data to display different things such as overall rating, author, etc. This feature could be really helpful for users and would be interesting to implement, although I wouldn't know where to start just yet.
